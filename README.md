新面板地址: [https://dashboard.locyanfrp.cn](https://dashboard.locyanfrp.cn)

## 如何使用

必须安装 Git 并初始化了仓库才能正常启动本项目

1. clone 本仓库

```shell
git clone https://github.com/LoCyan-Team/LoCyanFrpPanel.git
```

<!-- 2. 安装 [Node.js](https://nodejs.org) 18+ -->

2. 安装 Bun

请参阅: <https://bun.sh>

3. 安装依赖

```shell
bun i
```

4. 运行项目

```shell
bun dev
```
